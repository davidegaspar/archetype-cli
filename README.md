# archetype-cli

Model to create a Command Line Interface in NodeJS.

## Features

- logging
- commands
- flags
- prompts

```sh
arc -v
arc -d
arc -h
arc git-info # dotenv
arc demo # template files based on input
```

## Local development

```sh
npm install
npm link
arc
```

## Usage

```sh
# install
# npm install -g arc
arc
```
